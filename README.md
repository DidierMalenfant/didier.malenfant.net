### Installation

Install `rbenv`:
```
brew install rbenv ruby-build
```

Install a supported version of ruby:
```
rbenv install 3.2.1
rbenv global 3.2.1
```

Install dependencies:
```
gem install bundler jekyll
```

Install dependencies:
```
bundle install
```

Build the site:
```
bundle exec jekyll build
```

Start a web server at http://127.0.0.1:4000/:
```
bundle exec jekyll serve
```
